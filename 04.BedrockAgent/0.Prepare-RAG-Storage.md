RAG를 위해 Knowledge Base 문서를 업로드할 스토리지(S3)를 준비하고, 문서를 업로드 하겠습니다.

좌측 하단의 Open AWS console 버튼을 클릭하여 실습 계정으로 이동합니다.

us-east-1 리전에 있는지 확인합니다.
콘솔 -> S3로 이동

S3 버킷이름 :  s3://<본인의_AWS_계정번호>-rag
본인의_AWS_계정번호는 우측 상단에서 확인할 수 있습니다.

버킷이 만들어지면 "loandocs" 이란 이름의 폴더를 만듭니다.

해당 폴더에 [loandocs 링크](https://github.com/noenemy/q-cli-mcp/tree/main/04.BedrockAgent/loandocs)에 있는 PDF 파일을 업로드합니다.

Knowledge Base 문서와 스토리지가 준비 완료 되었습니다.
